Make any object viewable

## Install

Via Composer

``` bash
$ composer require funivan/viewable
```

## Usage

``` php

class Email {
 use \Funivan\Viewable\Viewable;
 
 public function getBody(){
  return $this->renderView('simple', array('name'=>'funivan'));
 }
 
}

```
